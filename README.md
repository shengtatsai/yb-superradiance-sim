# Yb Superradiance Simulator - public web build

Live site: https://shengtatsai.github.io/yb-superradiance-sim/

This repository contains only the compiled static site (no source code).
The site shows the laboratory's Yb superradiant-laser machine (3D CAD) with a
live physics simulation, plus the FEMTO-ST design for comparison.

femtost_cavity.glb is derived from the dat@UBFC dataset
doi:10.25666/DATAUBFC-2026-06-08 (Ruelle, Hauden, Ponciano-Ojeda, Delehaye),
licensed CC BY-SA.
